# Release Notes - v1.0.83

**Release Date:** 2026-01-14

## Highlights
This release enhances the onboarding experience with improved wizard functionality, better authentication UX, and more reliable tool installation scripts.

## New Features
- **Generalized Wizard Framework**: Added a flexible framework for tool installation wizards to support future extensibility
- **GitHub Authentication Improvements**:
  - Added OTP copy button for easier authentication flow
  - Added Reset button to restart authentication process
- **Enhanced Onboarding**: Improved wizard with password step, Homebrew support, and keyboard navigation

## Bug Fixes
- Fixed nvm/npm install script reliability issues
- Fixed context menu dismissal bug in GitHub authentication step
- Replaced deprecated `loadImageBitmap` with Skia Image API for better compatibility

## Improvements
- Better keyboard navigation in onboarding wizard
- More robust installation scripts for development tools
- Cleaner API usage by migrating from deprecated Skia methods

## Breaking Changes
None
