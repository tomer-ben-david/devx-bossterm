# Release Notes - v1.0.62

**Release Date:** 2025-12-28

## Highlights
This release addresses critical UI freeze issues that could occur during long terminal sessions and improves scrollbar visibility.

## Bug Fixes
- **UI Freeze Prevention**: Fixed UI freeze issues that could occur after long terminal sessions (issues #173, #179)
- **Scrollbar Width**: Increased default scrollbar width from 12px to 14px for better visibility and usability

## Breaking Changes
None
