# Release Notes - v1.0.39

**Release Date:** 2025-12-13

## Highlights
This release eliminates visual flickering issues during terminal output and improves the snap package build process with architecture-specific optimizations.

## Bug Fixes
- **TUI Flickering**: Eliminated flickering during PTY output by enabling adaptive debouncing
- **Input Flickering**: Fixed cursor flickering during streaming by making cursor non-reactive
- **Snap Version**: Updated snap version to be dynamically set in release workflow
- **Snap Architectures**: Limited snap build to amd64 and arm64 architectures for better reliability

## Breaking Changes
None
