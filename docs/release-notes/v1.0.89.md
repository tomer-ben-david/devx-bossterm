# Release Notes - v1.0.89

**Release Date:** 2026-01-29

## Highlights
This release introduces global hotkey support for summoning BossTerm windows and includes multiple fixes to improve reliability and stability of the hotkey implementation.

## New Features
- Add global hotkey to summon BossTerm windows (disabled by default on macOS and Linux)

## Bug Fixes
- Resolve critical bugs in global hotkey implementation across multiple rounds of fixes
- Fix native title bar toggle not persisting after restart
- Replace manual pointer event handling with declarative pointerHoverIcon API

## Improvements
- Disable global hotkeys by default on macOS and Linux for better platform compatibility

## Breaking Changes
None
