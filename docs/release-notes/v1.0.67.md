# Release Notes - v1.0.67

**Release Date:** 2026-01-02

## Highlights
This release enhances hyperlink detection and handling with improved URL pattern recognition, file path click support similar to iTerm2, and cross-platform compatibility improvements.

## New Features
- **File/Folder Path Click Support**: Added iTerm2-style clickable file and folder paths in terminal output (#204)
- **Rich Hyperlink Metadata**: Enhanced `onLinkClick` callback with `HyperlinkInfo` containing URL, type, bounds, and modifier keys
- **Fallback Behavior for Links**: Added fallback behavior for `onLinkClick` callback and documented `hyperlinkRegistry` usage

## Improvements
- **Selection System**: Follow-up improvements to text selection system (#204)
- **Starship Compatibility**: Set PWD environment variable to match `workingDirectory` for better Starship prompt compatibility

## Bug Fixes
- Fixed URL pattern to exclude trailing punctuation (periods, commas, etc.) from detected links
- Fixed `FilePathResolver` tests to be cross-platform compatible (Windows CI)

## Testing
- Added comprehensive unit tests for file path hyperlink detection

## Breaking Changes
None
