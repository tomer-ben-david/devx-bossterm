# Release Notes - v1.0.76

**Release Date:** 2026-01-06

## Highlights
This release introduces dynamic context menu customization with the new `contextMenuItemsProvider` API and adds async support for context menu callbacks.

## New Features
- **Dynamic Context Menu Items**: Added `contextMenuItemsProvider` parameter to `TabbedTerminal` and `EmbeddableTerminal` for programmatically customizing context menu items (#224)
- **Async Context Menu Callback**: Added `onContextMenuOpenAsync` callback as an alternative to `onContextMenuOpen` for asynchronous operations during context menu opening

## Improvements
- Enhanced context menu documentation with examples demonstrating `contextMenuItemsProvider` usage
- Updated examples to demonstrate both `onContextMenuOpen` and `onContextMenuOpenAsync` callbacks
- Added comprehensive KDoc documentation for `onInitialCommandComplete` and `onContextMenuOpen` callbacks

## Breaking Changes
None
