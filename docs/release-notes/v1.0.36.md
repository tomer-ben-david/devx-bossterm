# Release Notes - v1.0.36

**Release Date:** 2025-12-12

## Highlights

This release brings major improvements for Linux users with comprehensive distribution support (Snap, Deb, RPM), automatic updates, custom font support with iTerm2-style font picker, and session preservation across recomposition.

## New Features

- **Linux Distribution Support**: Added Snap, Deb, and RPM packages with automatic update support for Linux distributions
- **Custom Font Support**: Implemented iTerm2-style font picker for choosing custom terminal fonts (#103)
- **Session Preservation**: Terminal sessions now persist across recomposition, preventing data loss during UI updates (#104)
- **Symbol Font Configuration**: Added dropdown setting to control symbol font behavior with platform-specific defaults
- **Unfocused Window Indicator**: Visual overlay shows when terminal window is not in focus
- **Automated Dependency Updates**: Configured Dependabot for keeping dependencies up-to-date

## Bug Fixes

- Fixed Linux WM_CLASS handling using proper reflection for Java compatibility
- Added StartupWMClass to Linux desktop entry for proper icon display
- Resolved CLI script packaging issues by adding them to classpath resources
- Used pkexec for CLI installation on Linux for proper permissions
- Fixed incremental snapshot line tracking with IdentityHashMap
- Reverted macOS DMG naming to maintain backward compatibility
- Corrected snap publish conditions in release workflow

## Improvements

- Applied platform-specific font selection to all symbol types for better cross-platform rendering
- Bundled Noto Sans Symbols 2 for improved Linux symbol support
- Made symbol font setting nullable with intelligent platform defaults
- Restored original technical symbol behavior on macOS using system fonts
- Added CLI installation support via pkexec on Linux
- Auto-triggered JitPack builds on release
- Tracked BossTerm.iconset for proper icon regeneration

## Breaking Changes

None
