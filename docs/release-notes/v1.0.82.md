# Release Notes - v1.0.82

**Release Date:** 2026-01-11

## Highlights

This release brings comprehensive Windows platform support, including PowerShell/cmd.exe shell selection, package manager detection, and PTY improvements for a native Windows terminal experience.

## New Features

- Added Windows shell selection between PowerShell and cmd.exe
- Added Windows package manager prerequisite step to welcome wizard
- Added Windows support to welcome wizard and CLI installer

## Bug Fixes

- Improved Windows platform support and compatibility

## Improvements

- Centralized platform detection using ShellCustomizationUtils for better maintainability
- Enhanced Windows PTY implementation with multiple fixes and improvements
- Overall Windows platform improvements across the application

## Breaking Changes

None
